# Node.js basics

You can use following commands to check tasks:

    npm run fs:read
    npm run fs:copy
    npm run fs:create
    npm run fs:delete
    npm run fs:list
    npm run fs:rename
    npm run cli:args
    npm run cli:env
    npm run cp
    npm run calcHash
    npm run cjsToEsm
    npm run streams:read
    npm run streams:transform
    npm run streams:write
    npm run zip:compress
    npm run zip:decompress
    npm run wt
